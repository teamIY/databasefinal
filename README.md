# アプリケーション実行方法
1. githubリポリトジをクローン

```bash
    git clone https://github.com/teamIY/databasefinal.git
```

2. 依存関係をインストール
```bash
 cd databasefinal
 pip install flask
 pip install dotenv
 pip install datetime
 pip install sqlalchemy
 pip install logging
```
3. アプリケーション実行
```bash 
 python app.py
```
